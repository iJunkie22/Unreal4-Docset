FGraphDiffControl::FindNodeMatch

Syntax

static FNodeMatch FindNodeMatch
(
    class UEdGraph * OldGraph,
    class UEdGraphNode * Node,
    TArray< FNodeMatch > const & PriorMatches
)

Remarks

Looks through the supplied graph for a node that best matches the one specified. Sometimes (when diff'ing separate assets) there could be more than one possible match, so providing a list of already matched nodes helps us narrow it down (and prevents us from matching one node with multiple others).

A pair of nodes (including the supplied one) that best match each other (one may be NULL if no match was found).

Returns

A pair of nodes (including the supplied one) that best match each other (one may be NULL if no match was found).

Parameters

Parameter

Description

Graph

The graph you want to search.

Node

The node you want to match.

PriorMatches

Previous made matches to exclude from our search.
