SColorGradientEditor::DrawGradientStopMark

Syntax

void DrawGradientStopMark
(
    const FGradientStopMark & Mark,
    const FGeometry & Geometry,
    float XPos,
    const FLinearColor & Color,
    FSlateWindowElementList & OutDrawElements,
    int32 LayerId,
    const FSlateRect & InClippingRect,
    ESlateDrawEffect::Type DrawEffects,
    bool bColor,
    const FWidgetStyle & InWidgetStyle
)

Remarks

Draws a single gradient stop

Parameters

Parameter

Description

Mark

The stop mark to draw

Geometry

The geometry of the area to draw in

XPos

The local X location where the stop should be placed at

Color

The color of the stop

DrawElements

List of draw elements to add to

LayerId

The layer to start drawing on

InClippingRect

The clipping rect

DrawEffects

Any draw effects to apply

bColor

If true RGB of the Color param will be used, otherwise the A value will be used
