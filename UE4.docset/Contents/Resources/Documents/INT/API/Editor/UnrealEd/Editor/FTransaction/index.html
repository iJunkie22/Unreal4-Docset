Remarks

FTransaction.

A single transaction, representing a set of serialized, undo-able changes to a set of objects.

warning: The undo buffer cannot be made persistent because of its dependence on offsets of arrays from their owning UObjects.

warning: Transactions which rely on Preload calls cannot be garbage collected since references to objects point to the most recent version of the object, not the ordinally correct version which was referred to at the time of serialization. Therefore, Preload-sensitive transactions may only be performed using a temporary UTransactor::CreateInternalTransaction transaction, not a garbage-collectable UTransactor::Begin transaction.

warning: UObject::Serialize implicitly assumes that class properties do not change in between transaction resets.
