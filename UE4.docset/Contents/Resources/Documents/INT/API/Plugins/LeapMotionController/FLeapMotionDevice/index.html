FLeapMotionDevice

Syntax

class FLeapMotionDevice

Remarks

Implementation of the Leap Motion device. It bridges access to Leap API. You shouldn't use this in your code. Use LeapMotionFunctionLibrary, or LeapMotionControllerActor instead.
