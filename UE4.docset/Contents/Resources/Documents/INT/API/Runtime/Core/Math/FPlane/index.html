FPlane

Inheritance Hierarchy

FVector

FPlane

Syntax

struct FPlane : public FVector

Remarks

Structure for three dimensional planes.

Stores the coeffecients as Ax+By+Cz=D. Note that this is different than many other Plane classes that use Ax+By+Cz+D=0.
