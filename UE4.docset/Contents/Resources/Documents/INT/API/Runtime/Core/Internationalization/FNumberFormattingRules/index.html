NumberGroupSizes

Array to hold the number of digits belonging to groups of a number From NumberGroupSizes[0]..NumberGroupSizes[x] the number of digits are grouped for right to left after the NumberDecimalSeparator If the last element of the array is 0 then the remainder of the digits are not grouped otherwise the last element of the array is used as the size to group any remaining elements Examples English (UK) {3} => 123,456,789.00 Hindi {3,2} => 12,34,56,789.00 {3,0} => 123456,789.00
