TIndirectArray

Inheritance Hierarchy

TIndirectArray

TPanelChildren

Syntax

template<typename T, typename Allocator>
class TIndirectArray

Remarks

Indirect array. Same as a TArray above, but stores pointers to the elements, to allow resizing the array index without relocating the actual elements.
