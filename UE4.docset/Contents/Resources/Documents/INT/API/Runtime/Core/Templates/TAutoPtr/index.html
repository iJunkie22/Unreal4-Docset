TAutoPtr

Syntax

template<typename ReferencedType>
class TAutoPtr

Remarks

Wrapper around a raw pointer that destroys it automatically. Calls operator delete on the object, so it must have been allocated with operator new.

Same as TScopedPointer, except never calls new to make a duplicate
