ConfigFromIni

Syntax

struct ConfigFromIni

Remarks

Configuration helper classes and methods

A common way of configuring is from Inis, so this class supports that notion directly by providing a configuration class with a method suitable to be used as an FProviderConfigurationDelegate that reads values from the specified ini and section (based on the BuildType). Also provides a default location to store a provider name, accessed via GetDefaultProviderModuleName().
