Syntax

virtual void FinishDestroy()

Remarks

Called to finish destroying the object. After UObject::FinishDestroy is called, the object's memory should no longer be accessed.

note: because properties are destroyed here, Super::FinishDestroy() should always be called at the end of your child class's FinishDestroy() method, rather than at the beginning.
