Syntax

struct TStructOpsTypeTraitsBase

Remarks

Type traits to cover the custom aspects of a script struct

Constants

Name

Description

WithAddStructReferencedObjects

Struct has an ImportTextItem function used to deserialize a string into an object of that class.

WithCopy

Struct will not have its destructor called when it is destroyed.

WithExportTextItem

Struct can be compared via an Identical(const T* Other, uint32 PortFlags) function. This should be mutually exclusive with WithIdenticalViaEquality.

WithIdentical

Struct can be compared via its operator==. This should be mutually exclusive with WithIdentical.

WithIdenticalViaEquality

Struct can be copied via its copy assignment operator.

WithImportTextItem

Struct has an ExportTextItem function used to serialize its state into a string.

WithNetDeltaSerializer

Struct has a NetSerialize function for serializing its state to an FArchive used for network replication.

WithNetSerializer

Struct has a PostSerialize function which is called after it is serialized

WithNoDestructor

Struct has a constructor which takes an EForceInit parameter which will force the constructor to perform initialization, where the default constructor performs 'uninitialization'.

WithNoInitConstructor

Struct can be constructed as a valid object by filling its memory footprint with zeroes.

WithPostSerialize

Struct has a Serialize function for serializing its state to an FArchive.

WithSerializeFromMismatchedTag

Struct has a NetDeltaSerialize function for serializing differences in state from a previous NetSerialize operation.

WithSerializer

Struct has an AddStructReferencedObjects function which allows it to add references to the garbage collector.

WithZeroConstructor
