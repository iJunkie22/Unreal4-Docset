Syntax

virtual SIZE_T GetResourceSize
(
    EResourceSizeMode::Type Mode
)

Remarks

Returns the size of the object/ resource for display to artists/ LDs in the Editor. The default behavior is to return 0 which indicates that the resource shouldn't display its size.

Size of resource as to be displayed to artists/ LDs in the Editor.

Returns

Size of resource as to be displayed to artists/ LDs in the Editor.
