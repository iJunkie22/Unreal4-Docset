FMenuStack::PushMenu

Syntax

TSharedRef< SWindow > PushMenu
(
    const TSharedRef< SWindow > & InParentWindow,
    const TSharedRef< SWidget > & InContent,
    const FVector2D & SummonLocation,
    const FPopupTransitionEffect & TransitionEffect,
    const bool bFocusImmediately,
    const bool bShouldAutoSize,
    const FVector2D & WindowSize,
    const FVector2D & SummonLocationSize
)

Remarks

[Deprecated] Pushes a new menu onto the stack. Automatically removes windows from the stack that are no longer valid Invalid windows are those that are not parents of the window menu being pushed and any windows on the same stack level This is the old method for creating menus.

Parameters

Parameter

Description

InParentWindow

The parent window of the menu

InContent

The menu's content

SummonLocation

Location where the menu should appear

TransitionEffect

Animation to use when the popup appears

bFocusImmediately

Should the popup steal focus when shown?

bShouldAutoSize

True if the new window should automatically size itself to its content

WindowSize

When bShouldAutoSize=false, this must be set to the size of the window to be created

SummonLocationSize

An optional size around the summon location which describes an area in which the menu may not appear
