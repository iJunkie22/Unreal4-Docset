MouseCaptorHelper

Syntax

class MouseCaptorHelper

Remarks

A helper class to wrap the weak path functionality. The advantage of using this class is that the path can be validated and the current mouse captor (if any) can be informed they have lost mouse capture in the case of a number of events, such as application losing focus, or the widget being hidden.
