FScrollyZoomy

Syntax

class FScrollyZoomy

Remarks

Utility class that adds scrolling and zooming functionality to a widget. Derived your widget class from IScrollableZoomable, then embed an instance of FScrollyZoomy as a widget member variable, and call this class's event handlers from your own widget's event handler callbacks.
