IMediaInfo::SupportsRate

Syntax

bool SupportsRate
(
    float Rate,
    bool Unthinned
)

Remarks

Checks whether the specified playback rate is supported.

true if the rate is supported, false otherwise.

Returns

true if the rate is supported, false otherwise.

Parameters

Parameter

Description

Rate

The rate to check (can be negative for reverse play).

Unthinned

Whether no frames should be dropped at the given rate.
