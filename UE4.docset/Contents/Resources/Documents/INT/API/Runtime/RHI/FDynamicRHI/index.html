Remarks

============================================================================= DynamicRHI.h: Dynamically bound Render Hardware Interface definitions.

Statically bound RHI resource reference type definitions for the dynamically bound RHI.The interface which is implemented by the dynamically bound RHI.

Destructors

Name Description

~FDynamicRHI()

Declare a virtual destructor, so the dynamic RHI can be deleted without knowing its type.
