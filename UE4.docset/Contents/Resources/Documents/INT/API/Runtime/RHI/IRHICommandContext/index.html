IRHICommandContext

Inheritance Hierarchy

IRHICommandContext

FNullDynamicRHI

FOpenGLDynamicRHI

Syntax

class IRHICommandContext

Remarks

The interface RHI command context. Sometimes the RHI handles these. On platforms that can processes command lists in parallel, it is a separate object.
