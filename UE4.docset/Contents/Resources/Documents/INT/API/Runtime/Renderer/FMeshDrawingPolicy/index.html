FMeshDrawingPolicy

Syntax

class FMeshDrawingPolicy

Remarks

The base mesh drawing policy. Subclasses are used to draw meshes with type-specific context variables. May be used either simply as a helper to render a dynamic mesh, or as a static instance shared between similar meshs.
