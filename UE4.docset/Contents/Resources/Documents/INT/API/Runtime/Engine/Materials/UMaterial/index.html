Syntax

UCLASS(HideCategories=Object, MinimalAPI, BlueprintType)
class UMaterial : public UMaterialInterface

Remarks

A Material is an asset which can be applied to a mesh to control the visual look of the scene. In general, when light from the scene hits the surface, the shading model of the material is used to calculate how that light interacts with the surface.
