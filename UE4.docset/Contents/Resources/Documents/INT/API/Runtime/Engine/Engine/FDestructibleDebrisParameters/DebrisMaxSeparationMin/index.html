DebrisMaxSeparationMin

Syntax

UPROPERTY(EditAnywhere, Category=DestructibleDebrisParameters)
float DebrisMaxSeparationMin

Remarks

"Debris chunks" (see debrisDepth, above) will be destroyed if they are separated from their origin by a distance greater than maxSeparation. The actual maxSeparation is interpolated between these two values, based upon the module's LOD setting. To disable maxSeparation, clear the bDebrisMaxSeparation flag in the flags field. If debrisMaxSeparationMax < debrisMaxSeparationMin, the mean of the two is used for both. Default debrisMaxSeparationMin = 1.0, debrisMaxSeparationMax = 10.0f.
