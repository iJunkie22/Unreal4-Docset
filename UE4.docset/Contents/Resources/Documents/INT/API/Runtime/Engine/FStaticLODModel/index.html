FStaticLODModel

Syntax

class FStaticLODModel

Remarks

All data to define a certain LOD model for a skeletal mesh. All necessary data to render smooth-parts is in SkinningStream, SmoothVerts, SmoothSections and SmoothIndexbuffer. For rigid parts: RigidVertexStream, RigidIndexBuffer, and RigidSections.
